# Student Gradebook

Учебный проект для задания по продолжению начального курса Python.

Проект показывает использование аннотаций типов:

- типы аргументов и возвращаемых значений функций;
- `dict`, `list`, `Optional`;
- `dataclass` для описания данных;
- `Protocol` для интерфейса печати отчёта.

## Запуск

```bash
python3 main.py
```

## Тесты

```bash
python3 -m unittest discover -s tests -v
```

## Что делает программа

Программа хранит студентов, учебные курсы и оценки. Она умеет добавлять оценки,
считать средний балл студента и курса, а также формировать текстовый отчёт.
