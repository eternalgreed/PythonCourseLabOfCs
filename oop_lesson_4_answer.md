# Ответ на задание 4.2

В примере показаны параметрический полиморфизм и полиморфизм подтипов.

Параметрический полиморфизм я понял так: функция
`do_something_with_animal()` написана один раз и работает с параметром общего
типа `Animal`. Ей можно передать разные подходящие объекты, поэтому для каждого
дочернего класса не приходится писать отдельную функцию.

Полиморфизм подтипов проявляется при вызове `animal.foo()`. Объект хранится в
переменной общего типа `Animal`, но Python во время выполнения определяет его
настоящий класс. Поэтому для объекта `Cat` вызывается метод `Cat.foo()`, а для
объекта `Bird` - метод `Bird.foo()`.

В задании со списком программа заранее не проверяет, какой объект находится в
очередной позиции. Она одинаково вызывает `foo()` у каждого элемента, а нужная
реализация метода выбирается автоматически.
